# Ajay Sidal personal website

A premium, modern one-page personal brand site for Ajay Sidal built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Build

```bash
npm run build
```

## Visual system

- Palette: midnight, mist, ember, violet, cyan
- Mood: dark-first, editorial, premium, futuristic
- Motion: subtle and performance-conscious
- Structure: hero, about, strengths, experience, highlights, skills, education, personal side, contact

## Deployment

This project is deployment-ready for Vercel, Netlify, or any Node-compatible host.
